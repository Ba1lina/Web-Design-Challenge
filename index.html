PyCity

The purpose of the WWA was to analyse how weather changes according to where you are in terms of latitude. First, I generated a random selection longtitudes and latitudes. Using these random pairings I used PyCity to name the cities at these random coordinates. After this process I had 600+ cities to work with.

Open Weather Map

Using the random cities generated I pulled the weather data for those cities uding the Open Weather Map. The API used was for Current Data only.

Analysis

Once the data was gathered that is when analysis of the cities latitude vs other factors was done using python and jupyter notebook. The results of the Analysis can be found in the Visualisations pages and the data for the analysis can be found in the Data page.
